React-TypeScript-Electron-Three Визуализатор для см. МАРС
===========================================================================

## Установка доп. пакетов:

### `npm install`
### `yarn install`

## Запуск программы в режиме отладки:

### `npm run electron:dev`
### `yarn electron:dev`


## Компилирование программы:

### `npm run electron:build`
### `yarn electron:build`
